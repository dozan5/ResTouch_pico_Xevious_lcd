# ResTouch版 Xeviousもどき

[ResTouch](https://www.switch-science.com/products/7330)へ移植しました。  

　ResTouch版(VERTICALバージョン）

　ResTouch版(HORIZONTALバージョン）

uf2フォルダー内のファイルを参照ください。  
　　各HORIZONTAL,VERTICALフォルダーにpico版、pico2版に分けて保存しています。
